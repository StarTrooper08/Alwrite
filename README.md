# Alwrite
**Open Source note-taking and todo app**


#### Development setup for this project:

1. Fork this repository
2. clone this repository
`cmd` - git clone repo_url
3. cd Alwrite
4. npm install
5. npm start

Project will run on `localhost:3000`
